# **Alien Invasion** 

## **Descrição**
---
  **Projeto de um simples jogo de nave no modelo dos jogos arcade**

## **Controles**  
---
**Movimentação**
* W para mover pra cima
* S para mover para baixo
* A para a direita 
* D para a esquerda

**Disparos**
* SPACE para atirar os projetis
